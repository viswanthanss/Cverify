# Overview

🧠

CV Analysis

Upload a PDF — OCR + NLP classify the job role, XGBoost predicts hire probability and expected salary, KMeans assigns a candidate cluster.

🎯

Job Recommendation

TF-IDF hybrid model (cosine + Jaccard skills + salary) matches your CV against 20,414 real job postings and returns the top-5 most relevant.

🏢

Recruiter Portal

Recruiters post jobs, receive applications, and view AI-ranked candidate profiles with hire probability scores.

🤖

AI Chatbot

n8n-powered chatbot widget on every page. Answers career questions, guides users through CV submission, and provides personalized advice.

🕷️

Job Scraper

Admin-triggered scraper pulls live listings from RemoteOK, Arbeitnow and Keejob. Results auto-import into the DB with source tracking.

🔄

Full CI/CD

GitLab pipeline downloads Kaggle model artifacts, bakes them into Docker images, pushes to Docker Hub, and auto-deploys to the Kali VM.

# ML Pipeline

📄 PDF Upload pdfplumber + Tesseract OCR

Multi-page PDFs processed via pdfplumber with Tesseract OCR fallback (eng/fra/ara). Raw text extracted and fed to 3 parallel inference paths.

🏷️ Notebook 2 — NLP Job Role Classifier attiaimeed/nlp-model-comparison

TF-IDF (15k tokens, bigrams) + Random Forest (300 trees). 24 job categories. Tested against Logistic Regression, Linear SVC, Multinomial NB. F1-weighted: 75.5%. Exported as job_role_classifier.pkl.

📊 Notebook 1 — Candidate Evaluation attiaimeed/resume-ml-pj

Random Forest Classifier → hire probability (Acc: 100%). XGBoost Regressor → salary prediction (MAE: $21,106). KMeans k=2 → candidate cluster (Silhouette: 0.294). Exports: classifier_hire.pkl · salary_predictor.pkl · kmeans_clusters.pkl · label_encoder.pkl.

🎯 Notebook 3 — Job Recommender attiaimeed/job-recommender-model

Hybrid model: TF-IDF cosine similarity (0.5) + Jaccard skill overlap (0.3) + salary normalisation (0.2). Grid-optimised weights. Hit@5 / MRR@5 evaluation. Matches CV against 20,414 postings. Exported as job_recommender_v2.pkl.

# CI/CD Pipeline

Stage 1 — Models
📥 download-models
  • Install Kaggle CLI
  • Auth via KAGGLE_TOKEN env var
  • Download attiaimeed/resume-ml-pj
  • Download attiaimeed/job-recommender-model
  • Download attiaimeed/nlp-model-comparison
  • Verify all 3 required .pkl files exist
  • Pass backend/models/ as artifact
▼
Stage 2 — Build (parallel)
🐳 build-backend
  • Receives models artifact
  • docker build ./backend
  • Models baked into image
  • Push :SHA + :latest
🐳 build-frontend
  • No model dependency
  • Runs in parallel immediately
  • docker build ./frontend-angular
  • Push :SHA + :latest
▼
Stage 3 — Deploy (Kali VM runner, tag: deploy)
🚀 deploy
  • Shell executor on Kali VM
  • Runs $CI_PROJECT_DIR/deploy.sh
  • docker compose down (keeps volumes)
  • docker compose pull (new images)
  • docker compose up -d
